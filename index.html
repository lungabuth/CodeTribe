Mock Media Server API

This server allows you to manage movies, series, and songs using JSON-based requests.

Available Endpoints:

  • GET /movies
  • GET /series
  • GET /songs
  • POST /movies (add movie)
  • PUT /songs (update song)
  • DELETE /series (delete series by { "id": 1 })

Expected JSON:

{
  "title": "Summer House",
  "genre": "Crime",
  "year": 2019
}

All data is stored in a file (data.json) and updated in real-time.
